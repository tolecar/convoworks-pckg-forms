Search Entry Element

Searches entries by given value.

Parameters:

  • Context ID - ID of the referenced IAppointmentsContext
  • Search - The values you will use to search entries.
  • Return Variable Name - Default status, name of the variable that contains additional information (result)

Status variable fields

Fields for property Data Collection Item Name
Field Description
result The searched entries values which we found

Example of ${status.result} Return Variable Name value:

    {
        "field 1" : "value 1",
        "field 2" : "value 2",
        "entry_id" : "123"
    }

Flows:

  • Multiple flow - will be executed if the requested search has found multiple entries
  • Single flow - will be executed if the requested search has found a single entry
  • Empty flow - will be executed if the requested search didn't find anything
