Load Entry Element

Returns single entry data by an entry id.

Parameters:

  • Context ID - ID of the referenced IAppointmentsContext
  • Entry ID - ID of the existing appointment
  • Return Variable Name - Default status, name of the variable that contains additional information (entry, entry_id)

Example of ${status.entry} Return Variable Name value:

    {
        "field 1" : "value 1",
        "field 2" : "value 2",
        "entry_id" : "123"
    }

Flows:

  • OK - will be executed if the entry was found
  • Not Found - will be executed if the entry was not found
