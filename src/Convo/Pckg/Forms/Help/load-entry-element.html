Load Entry Element

Returns single entry by an entry id.

Parameters:

  • Context ID - ID of the referenced Form context
  • Entry ID - ID of the existing entry
  • Return Variable Name - Default status, name of the variable that contains additional information

Result variable fields

Fields for property Result Variable Name
Field Description
entry Loaded entry

Example of returned ${status.entry} as JSON:

    {
        "entry_id" : "123",
        "user_id" : "123",
        "meta_values" : {
            "key_1" : "value 1",
            "key_2" : "value 2"
        }
    }

Flows:

  • OK - will be executed if the entry was found
