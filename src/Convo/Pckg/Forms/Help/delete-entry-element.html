Delete Entry Element

Deletes an existing entry for the user.

Parameters:

  • Context ID - ID of the referenced IFormsContext
  • Entry ID - ID of the existing form entry
  • Result Variable Name - Default status, name of the variable that contains additional information (existing - previous form entry as you would get it with load entry element)

Status variable fields

Fields for property Data Collection Item Name
Field Description
existing The existing entry value which we will delete

Flows:

  • OK - will be executed when the entry was deleted
