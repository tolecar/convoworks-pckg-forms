Delete Entry Element

Deletes an existing entry for the user.

Parameters:

  • Context ID - ID of the referenced IFormsContext
  • Entry ID - ID of the existing form entry
  • Result Variable Name - Default status, name of the variable that contains additional information

Result variable fields

Fields for property Result Variable Name
Field Description
previous The existing entry value before deletion

Flows:

  • OK - will be executed when the entry was deleted
