Dummy Form Context

Example implementation of the Convoworms Forms functionality. It stores data in the User scope.

You can use this context to test your form based workflows or as an example when developing your own `FormsContext implementation.

Parameters

  • Context ID - Unique id for referencing this context
  • Required Fields - String, comma separated list of the fields which are required

Note on searching entries

When you are using this context with Search Forms Element parameter search is expected to be an associative array, containing field name and expected value pairs.
