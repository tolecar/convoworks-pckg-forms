Create Entry Element

This element will try to create an entry for given entry values.

Parameters:

  • Context ID - ID of the referenced IFormsContext
  • Payload - The entry data that will be written.
  • Result Variable Name - Default status, name of the variable that contains additional information (entry_id, entry)

Flows:

  • OK - executes when the entry could be created
  • Validation error - executes when the entry has validation errors
