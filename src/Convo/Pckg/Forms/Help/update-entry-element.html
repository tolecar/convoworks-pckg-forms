Update Entry Element

Element which updates existing entry value.

Parameters:

  • Context ID - ID of the referenced IFormsContext
  • Entry ID - ID of the existing entry
  • Entry - The entry data that will overwrite the previous data
  • Result Variable Name - Default status, name of the variable that contains additional information ( existing previous form entry as you would get it with load entry element, entry_id)

Result variable fields

Fields for property Result Variable Name
Field Description Flow
previous The existing entry before update OK
updated The entry after it has been updated OK
message String error message Validation error
errors Array of errors Validation error

Flows:

  • OK - will be executed when the entry is updated
  • Validation error - executes when the entry has validation errors
