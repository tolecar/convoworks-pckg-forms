Update Entry Element

Element which updates existing entry value.

Parameters:

  • Context ID - ID of the referenced IFormsContext
  • Entry ID - ID of the existing entry
  • Payload - The entry data that will overwrite the previous data
  • Result Variable Name - Default status, name of the variable that contains additional information ( existing previous form entry as you would get it with load entry element, entry_id)

Status variable fields

Fields for property Data Collection Item Name
Field Description
existing The existing entry value which we will update
updated The new value the entry has after updating
message Exception message
errors Error message

Flows:

  • OK - will be executed when the entry is updated
